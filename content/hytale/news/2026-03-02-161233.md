# Service Update

Version <span style="color:#55FF55">1.7.41</span> brings improvements to player shops and the auction house!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Shops & Auctions:</span>
      - Added a search bar so you can type to filter trades when browsing shops or previewing your own listings
      - New browse filters: "Can Afford", "Sell", and "Buy" to help you find trades faster; "Can Afford" checks your inventory and balance
      - When creating an auction you can set the starting bid and bid increment; bid increment is shown on auction cards
      - Shop blocks are protected by default so only the owner and co-owners can break them

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Shops & Auctions UI:</span>
      - Items now show rarity/quality backgrounds throughout shop and auction screens
      - Item tooltips use richer text for clearer info
      - Preview tabs now show item durability percentages like customers see
      - Large currency amounts display more neatly on cards and popups
      - You can enter much larger currency quantities when listing items (now up to 9,999,999)

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Shops:</span>
      - Shops set to unlimited stock now work correctly and will supply items and currency without getting stuck

Heads up: the new shop search, filters and adjustable auction bids are fresh additions and may have occasional issues. Please report anything odd at the Information Board.

Thanks to <span style="color:#55FFFF">makapar</span> for the improvements!
