# Service Update

Version <span style="color:#55FF55">1.7.75</span> brings NPC shop selling and nicer shop screens!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Shops:</span>
      - NPC shops (including buyer NPCs at the Town House) now let you sell items
      - Some NPCs open directly in SELL mode so you can sell right away

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Shop Screens:</span>
      - Shops now show separate BUY and SELL tabs; sell-only shops show only the SELL tab
      - SELL tab lists all sellable items, including ones you don't have (marked "Not in inventory")
      - Selling uses a confirmation window with a quantity selector and a total price display
      - Some shops may pay in a different currency

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Economy:</span>
      - Not all items can be sold; some items are excluded from selling
      - Daily sell limits per item have been added

Selling at NPC shops is a new feature and may have occasional issues. Please let us know if you find anything odd.

Thanks to <span style="color:#55FFFF">KyuubiDDragon</span> for the improvements!
