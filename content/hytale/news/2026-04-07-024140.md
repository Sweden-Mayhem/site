# Service Update

Version <span style="color:#55FF55">1.8.22</span> is live — small fixes and quality-of-life improvements!

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Combat:</span>
      - Parrying now works correctly with enchanted gear and includes a BETA visual and sound effect when an attack is incoming
   <span style="color:#FFAA00">Shops:</span>
      - Fixed an issue where items sold at player barter shops sometimes stayed in the shop chest
   <span style="color:#FFAA00">Mobs:</span>
      - Fixed problems that prevented some enemy variants from loading, so enemies should spawn and drop loot correctly now
   <span style="color:#FFAA00">HUD:</span>
      - Skill and combat HUD layout now saves correctly for new players

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Information Board:</span>
      - The Information Board (Help, Rules, Server Info, Team and search) now respects your selected language — 9 languages supported
   <span style="color:#FFAA00">Nameplates:</span>
      - Nameplates no longer stick to dead entities and are properly removed when you leave an area

<span style="color:#FFFF55">Performance</span>
   <span style="color:#FFAA00">Nameplates:</span>
      - Reduced workload for distant entities to improve server performance in busy areas

<span style="color:#FFFF55">Other Changes</span>
   <span style="color:#FFAA00">Changelog:</span>
      - Changelogs will now include small local server updates even when no other updates are pending

Note: Parry VFX/SFX are BETA and may have issues.
