# Service Update

Version <span style="color:#55FF55">1.8.19</span> is live — small improvements, smoother combat and clearer UI!

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Combat:</span>
      - Parrying now works correctly with enchanted weapons
      - Incoming-attack visual and sound cues added in BETA to help time perfect parries

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Information Board:</span>
      - The Information Board and its pages (Help, Rules, Server Info, Team, Search) now follow your selected language (9 languages supported)
   <span style="color:#FFAA00">Menus & HUD:</span>
      - Language choices now apply consistently across all menus and UI pages
   <span style="color:#FFAA00">Skill HUD:</span>
      - Skill and combat HUD now shows progress bars so you can track skill advancement in real time
   <span style="color:#FFAA00">Books & Mail:</span>
      - Books and the mail system gained additional language support
   <span style="color:#FFAA00">Nameplates:</span>
      - Fixed crashes and display problems with nameplates, especially for names with special characters; name labels should appear more reliably

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Barter Shops:</span>
      - Items sold through player shops now correctly leave the shop chest when sold

Note: The incoming-attack VFX/SFX is a BETA feature and may still have rough edges. Please report any odd behavior.
