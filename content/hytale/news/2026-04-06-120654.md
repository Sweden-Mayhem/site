# Service Update

Version <span style="color:#55FF55">1.8.21</span> brings a batch of fixes, quality improvements and a few new additions!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Worlds:</span>
      - Experimental Weekly World added - a teleporter to it is now located next to the spawn area
   <span style="color:#FFAA00">Website:</span>
      - Live World Map has been added to the website
      - New Play page added with server joining instructions

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Combat:</span>
      - Parrying now works correctly with enchanted weapons in more situations
      - Beta visual and audio cues added for incoming attacks to help time perfect parries
   <span style="color:#FFAA00">XP & Boosts:</span>
      - Two new XP boosts added to the Reliquary: Grand 10x and Epic 20x
   <span style="color:#FFAA00">Economy:</span>
      - Epic Repair Kit price adjusted to 5 Dark Coins at the Reliquary

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Shops & Marketplace:</span>
      - Player shops now properly remove sold items from chests when a sale completes
   <span style="color:#FFAA00">Enemies & Loot:</span>
      - Fixed several enemy loading issues so enemy variants spawn correctly and behave as intended
      - Dungeon Goblin loot adjusted: weapon, armor and offhand equipment no longer drop from that instance
      - Cultist family XP rewards fixed - Cult Knights, Werewolves and bosses now grant appropriate XP
   <span style="color:#FFAA00">Weapons & Effects:</span>
      - Azaroth Longsword charged attack now shows its visual and audio effect when it connects
   <span style="color:#FFAA00">HUD & Nameplates:</span>
      - Skill/combat HUD layout now saves correctly for new players
      - Nameplates cleaned up and performance improved so dead or distant entities no longer leave stray nameplates
   <span style="color:#FFAA00">Books & Mail:</span>
      - Fixed crash when sending mail with a full inventory
      - Lectern added so books can be showcased and copied
      - Mail send tooltip clarified and the mailbox crafting recipe restored
   <span style="color:#FFAA00">Loot Boxes & Languages:</span>
      - Fixed a French language file issue that could affect loot box text display

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Information Board & Languages:</span>
      - The Information Board and its pages now respect your selected language across the interface
      - Extra unused language files were removed and team rank descriptions were corrected to English
   <span style="color:#FFAA00">Website:</span>
      - Live World Map and Play page are available on the website for quick reference

Note: The Weekly World is experimental and may have issues. Report any problems you find on the Information Board.
