# Service Update

Version <span style="color:#55FF55">1.7.79</span> brings new building blocks and visual upgrades!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Building:</span>

      - Glass types now visually connect so large windows and panes form seamless surfaces
      - You can change a placed glass block to a different glass type using a hammer

These glass pieces are new and may have minor visual quirks. Please report anything odd you find.
