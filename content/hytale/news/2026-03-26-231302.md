# Service Update

Version <span style="color:#55FF55">1.7.86</span> brings new dungeon rewards, mounts and several updates!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Dungeons:</span>
      - Boss loot now comes from Treasure Bags awarded to everyone who damaged the boss; only the dungeon exit key drops directly
      - The Nameless God now sells a Silver Moon Treasure Bag that contains Silver Moon items and a chance at an Epic Mount
      - Some chests can be Mimics now - be careful when looting
      - New bench added: the Mjolnir Furnace, used to make magical materials and smelt artifacts

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Mounts:</span>
      - Spectral Stones can now summon mounts when used; these stones are found in Golden Chests and Treasure Bags
      - Mount rarities now affect speed and jump: Legendary (fastest, high jump), Epic (very fast), Rare (faster), Uncommon (vanilla-like)
      - Available mounts and how to get them:
         - Brown Horse (Uncommon) - found in Chests/Treasure Bags
         - Armored Horse (Black) - Rare, craftable in the Mjolnir Furnace (two armor styles)
         - Black Wolf (Uncommon) - found in Chests/Treasure Bags
         - White Wolf (Uncommon) - craftable in the Mjolnir Furnace
         - Hell Wolf (Red, Epic) - craftable in the Mjolnir Furnace
         - Silver Moon Wolf Spirit (Purple, Epic) - only from the Silver Moon Treasure Bag

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Combat:</span>
      - Parry system updated for the latest game version; you may encounter small issues while we watch for fixes
   <span style="color:#FFAA00">Training Dummies & Gliders:</span>
      - Training dummies updated so they behave correctly
      - Gliders updated and a few issues fixed

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Item Viewer:</span>
      - Item and recipe viewer updated for compatibility so tool and weapon info displays correctly
