# Service Update

Version <span style="color:#55FF55">1.8.19</span> brings quality-of-life updates, language support, and a few combat and shop fixes!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Combat:</span>
      - Added BETA visual and sound cues for incoming attacks to help time perfect parries
   <span style="color:#FFAA00">Skills HUD:</span>
      - Skill HUD now shows progress bars so you can track skill progress more clearly
   <span style="color:#FFAA00">Books & World:</span>
      - A lectern has been added so books can be showcased and copied
   <span style="color:#FFAA00">Languages:</span>
      - The Information Board and related pages now support 9 languages
      - Books and written content gained additional language support

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Parrying:</span>
      - Perfect parries now work correctly with certain enchantments
   <span style="color:#FFAA00">Shops & Economy:</span>
      - Player barter shops now properly remove items from chests when sold
   <span style="color:#FFAA00">Enemies:</span>
      - Fixed issues where some enemy variants failed to load or gave incorrect drops; mobs should spawn and drop loot correctly again
   <span style="color:#FFAA00">Mail & Books:</span>
      - Fixed a crash that could happen when sending mail with a full inventory
      - Restored the missing mailbox recipe and clarified the send-mail tooltip
   <span style="color:#FFAA00">User Interface:</span>
      - Language selection now applies across the Information Board and related menus instantly, so Help, Rules, Server Info, Team and Search show in your chosen language

Note: the incoming-attack VFX/SFX is marked BETA and may still have occasional issues.
