# Service Update

Version <span style="color:#55FF55">1.8.23</span> brings improvements to combat, shops, language support and the UI!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Combat:</span>
      - New BETA visual and sound cue for incoming attacks in the parry system
   <span style="color:#FFAA00">Language:</span>
      - Language selection added (9 languages). Language changes now apply instantly across menus and NPCs

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Shops:</span>
      - Player shops and auction chests now remove sold items correctly
   <span style="color:#FFAA00">Combat:</span>
      - Parry system now works correctly with enchantments
   <span style="color:#FFAA00">Stability:</span>
      - Fixed a recurring crash that could cause disconnects and instability
   <span style="color:#FFAA00">Skill HUD:</span>
      - Skill and combat HUD layout now saves correctly for new players

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Mobs:</span>
      - Enemy variants and loot now load correctly so mobs behave and drop items as intended
      - Temporary debug markers on mobs are now removed when debug mode is turned off

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Nameplates & Performance:</span>
      - Nameplates no longer stick to dead creatures or remain visible after leaving an area
      - Reduced lag in busy areas by skipping nameplate updates when no players are nearby

Some features are new or BETA (notably the parry VFX/SFX), so you may spot small quirks. Thanks for testing and reporting bugs!
