# Service Update

Version <span style="color:#55FF55">1.8.22</span> brings several fixes and improvements!

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Combat:</span>
      - Perfect parry timing now works correctly with enchanted gear
      - New BETA visual and audio cue when an attack is incoming to help time parries (may be rough)
   <span style="color:#FFAA00">Shops:</span>
      - Selling from player barter shops now correctly removes the sold items from your chest
   <span style="color:#FFAA00">Mobs:</span>
      - Several enemy spawn and loot issues have been fixed so mobs behave and drop correctly
      - Any stray debug nameplates are now cleared from enemies

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Information Board:</span>
      - Language selection added (9 languages). Help, Rules, Server Info, Team and Search will follow your chosen language
      - Language changes now apply instantly across all server menus and NPC dialogs
   <span style="color:#FFAA00">HUD:</span>
      - Skill and combat HUD layout now saves properly for new players
      - "What you're looking at" HUD now works correctly alongside other HUDs
   <span style="color:#FFAA00">Item Info:</span>
      - Item info tooltips redesigned for clearer, cleaner display

<span style="color:#FFFF55">Performance</span>
   <span style="color:#FFAA00">Nameplates:</span>
      - Nameplates and entity info have been optimized to reduce lag with many mobs loaded
      - Nameplates now clean up correctly when entities die or players leave the area

Note: the incoming attack VFX/SFX for parries is a BETA feature and may still have issues.
