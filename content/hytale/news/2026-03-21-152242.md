# Service Update

Version <span style="color:#55FF55">1.7.82</span> brings changes to combat and the parry system!


<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Combat:</span>
      - Enemies can now parry players and other enemies, so fights will feel more reactive.
      - Reminder: a well-timed parry still avoids stamina loss, reduces incoming damage, and can lead to a stun when the attacker runs out of stamina.


<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Parry UI:</span>
      - Parry interface updated for clearer navigation and timing indicators so it is easier to see when to parry.

Short update — expect combat to feel more tactical in fights.

Thanks to <span style="color:#55FFFF">narwhals</span> for the improvements!
