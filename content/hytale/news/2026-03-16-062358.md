# Service Update

Version <span style="color:#55FF55">1.7.72</span> brings smoother menus and easier inputs!

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Menus:</span>
      - Buttons across the Information Board and other in-game menus are clearer and easier to use
      - Button text now scales better so labels stay readable
      - The old "Loading page" placeholder has been removed for a cleaner menu open
   <span style="color:#FFAA00">Input Fields:</span>
      - Input boxes are bigger, making typing names and numbers easier
      - Helpful preview text now appears in input fields so you know what to enter
      - You will now get a clear message if a required input field is left empty
