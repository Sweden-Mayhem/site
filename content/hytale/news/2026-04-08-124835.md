# Service Update

Version <span style="color:#55FF55">1.8.23</span> brings quality fixes and a few nice improvements!

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Combat:</span>
      - Parrying now works correctly with enchanted gear
      - New incoming-attack VFX/SFX added (BETA)
   <span style="color:#FFAA00">Shops & Economy:</span>
      - Selling from player barter shops now properly removes items from the chest
   <span style="color:#FFAA00">Mobs:</span>
      - Many enemy issues fixed so mobs load and behave correctly and drops work as expected
      - Debug nameplates that could stick to mobs have been removed

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Information Board & Language:</span>
      - The Information Board, Help, Rules, Team and Search pages now respect your chosen language (9 languages available)
      - Language changes now apply consistently across all in-game menus
   <span style="color:#FFAA00">Skill HUD:</span>
      - Skill HUD layout now saves correctly for new players
   <span style="color:#FFAA00">Nameplates:</span>
      - Nameplates are more efficient and will clean up correctly when mobs die or players leave the area
   <span style="color:#FFAA00">Item Info:</span>
      - Item info tooltips received a cleaner, easier-to-read design
