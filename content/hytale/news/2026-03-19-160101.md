# Service Update

Version <span style="color:#55FF55">1.7.80</span> brings smoother, more polished gliders!

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Gliders:</span>
      - Gliders feel smoother and more responsive when you start and control a glide
      - All gliders have adjusted fall speeds for more natural flight
      - Riptide glider performance increased for stronger lift and speed
      - Magma glider performance increased for better handling
      - Void glider slightly weakened for better balance
      - Volt glider has new visuals, particles and sounds for a flashier experience
      - A new sound now plays when you try to use a glider in an area where gliding is blocked

Thanks to <span style="color:#55FFFF">NikaidouShinku</span> for the improvements!
