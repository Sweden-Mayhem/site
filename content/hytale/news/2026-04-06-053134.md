# Service Update

Version <span style="color:#55FF55">1.8.21</span> brings a batch of fixes, quality improvements and a few new additions!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Worlds:</span>
      - Experimental Weekly World added - teleporter placed next to the spawn area (may be unstable)
   <span style="color:#FFAA00">Website:</span>
      - Live World Map added to the website
      - New Play page added to the website with server joining instructions

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Shops & Auction:</span>
      - Player-run barter shops now correctly remove sold items from shop chests
   <span style="color:#FFAA00">Reliquary:</span>
      - Two new XP boosts added: Grand 10x and Epic 20x
      - Epic Repair Kit price adjusted to 5 Dark Coins at the Reliquary
   <span style="color:#FFAA00">Dungeons:</span>
      - Goblin dungeon loot adjusted - weapons, armor and offhand items removed from Goblin drops
   <span style="color:#FFAA00">Mobs:</span>
      - Some mob variants that previously failed to load or had missing loot should now spawn and behave correctly

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Information Board:</span>
      - Language support improved so pages respect your selected language
      - Language options streamlined and team rank descriptions fixed to English on the Information Board
   <span style="color:#FFAA00">Skill HUD:</span>
      - Skill/combat HUD layout now saves correctly for new players
   <span style="color:#FFAA00">Nameplates:</span>
      - Nameplates are more responsive and no longer remain on dead or distant entities

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Books & Mail:</span>
      - Fixed a crash when sending books with a full inventory
      - Lectern added so you can showcase and copy books
      - Mailbox recipe restored and send-mail tooltip clarified; you can now send empty books if you want
   <span style="color:#FFAA00">Loot Boxes:</span>
      - Fixed a French language text error for loot boxes

Try the new Weekly World if you dare - it is experimental and may have issues.
