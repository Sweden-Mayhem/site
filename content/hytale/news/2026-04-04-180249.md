# Service Update

Version <span style="color:#55FF55">1.8.19</span> is live — small improvements and fixes!

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Combat:</span>
      - Perfect parry now works correctly with enchanted gear
      - BETA: visual and sound cues added to warn you when an attack is incoming (may still have issues)
   <span style="color:#FFAA00">Shops & Auction:</span>
      - Player-run shops now properly remove items from shop chests when they are sold
   <span style="color:#FFAA00">Enemies:</span>
      - Enemy variants and their loot now load correctly, so enemies and drops should appear more reliably
   <span style="color:#FFAA00">Books & Mail:</span>
      - Lectern added so you can showcase and copy books
      - Mailbox recipe fixed and send-mail tooltip clarified
      - Fixed crashes when opening books with a full inventory; non-ASCII characters now display correctly
      - You can now send empty books if you want

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Information Board & Menus:</span>
      - Language support added (9 languages); Help, Rules, Server Info and other pages now follow your selected language across menus and NPC text
   <span style="color:#FFAA00">Skill HUD:</span>
      - Skill HUD now shows progress bars for skills
