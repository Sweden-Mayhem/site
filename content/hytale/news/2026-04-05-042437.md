# Service Update

Version <span style="color:#55FF55">1.8.19</span> is live — small fixes and a few neat additions!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Combat:</span>
      - BETA visual and sound cue added for incoming attacks to help with timing perfect parries
   <span style="color:#FFAA00">HUD:</span>
      - Skill HUD now shows progress bars so you can track skill progress at a glance
   <span style="color:#FFAA00">Books:</span>
      - A lectern is now available to showcase and copy books

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Shops:</span>
      - Player barter shops now correctly remove sold items from chests after a sale
   <span style="color:#FFAA00">Combat:</span>
      - Perfect parry now works correctly when using enchanted gear
   <span style="color:#FFAA00">Mobs & Loot:</span>
      - Several enemy variants and their loot have been fixed so mobs load and drop correctly
   <span style="color:#FFAA00">UI & Language:</span>
      - Information Board pages (Help, Rules, Server Info, Team, Search) now respect your selected language consistently
   <span style="color:#FFAA00">Books:</span>
      - Fixed crashes caused by full inventories when sending books; mailbox recipe fixed; you can now send empty books

<span style="color:#FFFF55">Performance</span>
   <span style="color:#FFAA00">Nameplates:</span>
      - Nameplates now skip distant entities to reduce lag, and nameplates for dead entities are cleaned up properly

Some features are new or marked BETA and may still have rough edges. Report anything odd you find at the Information Board.
