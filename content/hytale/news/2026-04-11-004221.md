# Service Update

Version <span style="color:#55FF55">1.8.28</span> brings cleaner nameplates and stability fixes!

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Nameplates:</span>
      - Pets and some other entities now display their assigned name instead of showing their entity type

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Nameplates:</span>
      - NPC names no longer include internal suffixes like Patrol, Wander, Guard, or Idle (for example "Skeleton Fighter" instead of "Skeleton Fighter Wander")
      - Adjusting the vertical position of nameplates no longer causes the game to crash

Enjoy the clearer nameplates!
