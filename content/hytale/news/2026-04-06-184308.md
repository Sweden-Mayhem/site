# Service Update

Version <span style="color:#55FF55">1.8.21</span> brings fixes and quality-of-life improvements!

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Combat:</span>
      - Perfect parries now work correctly with enchantments
      - Incoming-attack VFX and SFX added (BETA) to help time perfect parries
   <span style="color:#FFAA00">Player Shops:</span>
      - Barter shops now correctly remove sold items from shop chests so trades no longer leave duplicates
   <span style="color:#FFAA00">Enemies:</span>
      - Fixed multiple issues that could cause some custom enemies to fail to load or drop incorrect loot; mob variety and loot should be more reliable

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Information Board:</span>
      - The Information Board (Help, Rules, Server Info, Team, Search) now supports 9 languages and shows pages in your chosen language
   <span style="color:#FFAA00">Language Support:</span>
      - Language selection now applies instantly across NPCs and in-game menus
   <span style="color:#FFAA00">Skill HUD:</span>
      - Skill and combat HUD layout now saves correctly for new players
   <span style="color:#FFAA00">Nameplates:</span>
      - Nameplate updates reduce lag when many entities are nearby and prevent nameplates from sticking to dead mobs

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Server Stability:</span>
      - Prevented a crash on restart that could duplicate Books and Papers

Note: Parry VFX is a BETA feature and may still have rough edges. Expect tweaks.
