# Service Update

Version <span style="color:#55FF55">1.7.71</span> brings loot improvements and fixes!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Loot Chests:</span>
      - Areas that previously had no assigned loot will now contain a random default set of items in their chests

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Loot Behavior:</span>
      - Fixed cases where chests in flat or specially generated areas (for example, some temples or starter zones) did not give loot correctly

Note: The new default loot for previously-empty areas may be adjusted in future updates.
