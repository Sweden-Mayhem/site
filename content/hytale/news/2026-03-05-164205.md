# Service Update

Version <span style="color:#55FF55">1.7.47</span> brings language options and important shop fixes!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Languages:</span>
      - Pick your preferred language for UI, HUD and chat messages.
      - You can choose different languages per feature so different parts of the game can use different languages.
      - Available languages include English, German, French, Spanish, Portuguese (BR), Russian, Polish, Turkish and Italian.

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Shops:</span>
      - Daily rotation shops now reset correctly each day.
      - Purchase limits now reset daily for all shop types.
      - Daily shop items that went missing are now regenerated so shops should no longer show empty or broken entries.

<span style="color:#FFFF55">User Interface</span>
   <span style="color:#FFAA00">Display:</span>
      - Additional UI colors added for clearer displays.

Language selection is a new feature and may still have a few small issues. Thanks to <span style="color:#55FFFF">KyuubiDDragon</span> for the improvements!
