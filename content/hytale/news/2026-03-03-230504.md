# Service Update

Version <span style="color:#55FF55">1.7.43</span> brings updates to enemies, loot and combat behavior!

<span style="color:#FFFF55">New</span>
   <span style="color:#FFAA00">Mobs:</span>
      - Summoned reinforcements now match their summoner's faction, so fights feel more consistent
   <span style="color:#FFAA00">Loot:</span>
      - Mob drops have been improved and now vary more reliably by tier
   <span style="color:#FFAA00">Effects:</span>
      - New status effects can appear on enemies, adding more variety to encounters

<span style="color:#FFFF55">Gameplay</span>
   <span style="color:#FFAA00">Abilities:</span>
      - Mobs now use abilities appropriate to the weapon they hold (for example, staff users will stop using melee-only leaps)
      - Higher-tier versions of a mob can have different abilities than lower-tier ones
   <span style="color:#FFAA00">Elites:</span>
      - Elites are less likely to attack each other, making elite fights clearer and less chaotic

<span style="color:#FFFF55">Bug Fixes</span>
   <span style="color:#FFAA00">Appearance:</span>
      - Fixed cases where mobs looked wrong because they wore armor in unsupported slots
   <span style="color:#FFAA00">Behavior:</span>
      - Fixed pickaxe-wielding mobs behaving like axe users

Some of these changes add new behavior to enemies and loot, so you may notice differences in a few areas while we keep polishing things.

Thanks to <span style="color:#55FFFF">Frotty277</span> for the improvements!
